# MCP Extensions

This directory contains the MCP (Model Context Protocol) extensions infrastructure for the StrategicKhaos Swarm Intelligence ecosystem.

## 🏗️ Architecture

The MCP Extensions suite provides modular integrations for various aspects of the sovereign AI infrastructure:

```
mcp-extensions/
├── refinery/          # CLI to create/test/deploy extensions
├── queen/             # Queen API integration (Port 8080)
├── treasury/          # Financial signals and tracking
├── academic/          # SNHU email processing
└── sovereign/         # Full empire access and control
```

## 📦 Modules

### Refinery
CLI tooling for creating, testing, and deploying MCP extensions.

### Queen
Integration with the Queen API running on port 8080. Provides health checks and API communication.

### Treasury
Financial signals processing and tracking for the StrategicKhaos ecosystem.

### Academic
SNHU email processing and academic integration tools.

### Sovereign
Full empire access module for comprehensive system control.

## 🚀 Quick Start

Each module contains its own README with specific setup and usage instructions.

## 🔧 Configuration

Modules are designed to work with:
- GitHub Codespaces (port forwarding)
- Custom MCP Agents (Strategickhaos-Swarm-Intelligence organization)
- Enterprise GitHub features

## 📝 Prerequisites

- GitHub Enterprise account with StrategicKhaos Swarm Intelligence
- Custom MCP Agents enabled (Set Organization in GitHub Copilot settings)
- Updated billing information for Enterprise features
- Codespace with port 8080 forwarded for Queen API

## 🌐 Deployment

See individual module READMEs for deployment instructions.

## 🔐 Security

- All API endpoints require proper authentication
- Enterprise features require valid GitHub Enterprise subscription
- Custom agents restricted to Strategickhaos-Swarm-Intelligence organization
